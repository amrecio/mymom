The main package of the ai package tree.

The main AI-class is {@link net.sf.freecol.server.ai.AIMain}.

The super class of the AI-objects that are linked to a FreeColGameObject is {@link net.sf.freecol.server.ai.AIObject}.

The main class for a given AI-player is {@link net.sf.freecol.server.ai.AIPlayer}. When it is an AI-player's turn, the method {@link net.sf.freecol.server.ai.AIPlayer#startWorking} gets called for each AI-controlled player.
